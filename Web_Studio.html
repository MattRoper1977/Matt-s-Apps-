How to build your page

  1. page.html is what's on the page; style.css is how it looks. The preview updates as you type.
  2. Tap the buttons below the editor to drop in real code — headings, pictures, lists, links, colours.
  3. 🖼 Add picture imports a photo and gives it a short name like pic1 — your saved page carries the picture inside it.
  4. Try your page at 📱 phone size — does it still look good?
  5. 💾 Save my page downloads a real .html file. Open it anywhere — that's your website, made by you.
Safety note for curious coders: <script> is switched off in the preview — this studio is for HTML and CSS. Your work autosaves on this device.
