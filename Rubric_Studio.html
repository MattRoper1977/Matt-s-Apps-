How to use Rubric & Feedback Studio

  1. Build a rubric: name it, set your level names (e.g. Emerging → Mastered), and add criteria with a short descriptor for each level.
  2. Add your pupils, then for each one tap the level that fits each criterion.
  3. It works out an overall grade and writes structured feedback (What went well / Even better if / Next steps) from the levels you chose.
  4. Tweak the wording, tap comment-bank phrases to add more, then 📋 Copy or 📄 Save PDF a clean feedback sheet.
Feedback is strengths-first and constructive by design. Build a rubric once and reuse it for the whole class. Everything stays on this device.
