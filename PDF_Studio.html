How to use PDF Studio

  1. Open a PDF with the ＋ Open button.
  2. Pick a tool: Pen to write, Highlight, Text to type, Stamp for ✓ ✗ ★ and feedback badges.
  3. Made a mistake? Undo ↩ or use the Eraser.
  4. Use the Select tool to move or resize things, or drag empty space to scroll.
  5. ✏ Edit text — tap any line of the document's own text to retype it. The original is covered and your words drawn on top in a matching colour; the Eraser brings the original back. Note: the old text still exists inside the file underneath, so don't use this to hide sensitive information.
  6. Pages tab: reorder, rotate ⟳, duplicate ⧉, delete, or add another PDF to merge.
  7. Export ⬇ saves your finished PDF to downloads.
Teacher tips: fill forms with the Text tool + ✓ stamps. Split a document by ticking pages in the Pages tab, then "Save selected as PDF". WWW / EBI / VF badges live in the Stamp tray. Typed text exports in standard letters — emoji won't print.
