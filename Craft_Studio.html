How to code the bot

  1. Tap the code buttons to drop real commands into your program — or just type them. move(3) walks three squares; turnLeft() and turnRight() spin the bot.
  2. place("stone") puts a block on the square in front; mine() digs one away. Place onto the same square again and again to build a tower.
  3. repeat 4 { … } runs the bit inside four times — the secret to short, clever programs. if (blocked) { … } only runs when there's a wall in front.
  4. The bot climbs one block up, drops any distance, bonks off tall walls, and can't swim — bridge water with planks.
  5. ▶ Run plays your program line by line. Step on the gold and you win instantly. 📷 Evidence saves your code and your build as one picture.
Your code is saved on this device for every level, so you can pick up where you left off. Ctrl+Enter also runs.
