How to make music

  1. Each row is a sound; each square is a beat. Tap to switch a square on. Tap again for a longer/accented note, once more to clear.
  2. Drums keep time. Bass, Melody and Lead play notes (higher row = higher note). Chords play three notes at once for instant harmony.
  3. Pick a Key and Scale up top — pentatonic scales mean there are no wrong notes. Set each track's fader, mute 🔊 or solo ★.
  4. Build different patterns (A, B, C, D) using the tabs, then in Song add them in order to make a full arrangement.
  5. 💾 Save WAV renders the whole song (or one pattern) to an audio file for Video Studio. It's all built on the device — 100% yours, no copyright.
